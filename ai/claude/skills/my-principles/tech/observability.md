# Наблюдаемость

> Принципы логирования, трейсинга, метрик. Общие принципы — в [SKILL.md](../SKILL.md).

## Инструменты

- **OpenTelemetry напрямую** — без лишних обёрток.

## Принципы

- **Прямой OTel,** без самописных абстракций поверх.
- **Ошибки наружу — полный traceback,** не один `msg`.

## Анти-паттерны

- Обёртки поверх OTel без выгоды.
- Глотание ошибок; логирование одного `msg` вместо traceback.

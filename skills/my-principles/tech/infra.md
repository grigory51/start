# Инфра и dev-окружение

> Принципы инфраструктуры, локальной разработки и деплоя. Общие принципы — в [SKILL.md](../SKILL.md).

## Инструменты

- **Контейнеры: Docker / docker-compose.**
- **Локальная разработка: Tilt.**
- **Деплой: serverless (Yandex Cloud).** Важна быстрая загрузка контейнера — оптимизируй cold start.
- **Монорепо: Yandex Arc.** *(контекст конкретного проекта.)*

## Принципы

- **Без костылей.** Есть рекомендованный способ — переезжать на него («чтобы без костылей»).
- **Быстрый cold start** для serverless: лёгкие образы, минимум на старте.
- **Cache invalidation по версии.** Не держать устаревший кеш после обновления.

## Анти-паттерны

- Костыли и обходные пути там, где есть рекомендованный способ.
- Дублирование конфигов между окружениями.
- Тяжёлые образы, медленный cold start на serverless.
